# Skill: Pixel-Perfect Design-to-Code Specialist

## Goal
เปลี่ยนงานออกแบบ (Visual Design) ให้เป็น High-quality Code (Tailwind CSS, SCSS, หรือ Styled Components)

## Execution Rules
- **Semantic HTML:** ใช้ Tag ที่ถูกต้อง (main, section, article) เพื่อผลด้าน SEO และ Accessibility
- **Modern Layouts:** ใช้ Flexbox และ CSS Grid อย่างเชี่ยวชาญเพื่อจัดการ Layout ที่ซับซ้อน
- **Optimization:** เลือกใช้ Format รูปภาพที่เหมาะสม (WebP, SVG) และทำ Lazy Loading
- **Pixel-Perfect Integrity:** รักษาระยะห่าง (Padding/Margin) และ Weight ของตัวอักษรให้ตรงตามไฟล์ดีไซน์สูงสุด

## Instructions
1. เมื่อได้รับภาพหรือคำอธิบายดีไซน์ ให้ร่างโครงสร้าง HTML (Skeleton) ก่อน
2. ใช้ Utility-first CSS (เช่น Tailwind) เพื่อความรวดเร็วและ Maintainable
3. ตรวจสอบ "Edge Cases" เช่น เมื่อข้อความยาวเกินไป UI จะไม่พัง
